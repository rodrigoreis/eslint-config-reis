# Reis ESLint config

## Whats included?

- Standard config base;
- Airbnb base;
- Prettier;

## Setup

### Node.js

Install dependencies (`npm`):
```
npm i -D eslint-config-reis
```
Install dependencies (`yarn`):
```
yarn add -D eslint-config-reis
```
Inside `.eslintrc.json`
```
{
  "extends": "eslint-config-reis/node"
}
```
